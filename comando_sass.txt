GIT 
git init -> crear repositorio en git
git add . -> subir cambios desde "Working Copy" a la rama "Staging" 
git log
git log --oneline 
git commit -m "mensaje" -> subir los cambios al repositorio (rama actual)
git branch otra_rama -> crear otra_rama
git checkout otra_rama -> me paro en la rama "otra rama"
git diff otra_rama -> muestra las diferencias entre la rama actual y la rama otra_rama
git merge otra_rama -> para hacer el merge
